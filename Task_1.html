Basic HTML Tags

HTML (Hypertext Markup Language) is the standard language for creating web pages. This section covers some basic HTML tags.

Meta Tags

Meta tags provide metadata about an HTML document. They are always placed in the <head> section of the HTML document. Example meta tags include charset, author, viewport, and description.
